Чудесный праймер

Очиститель Weld-On® C-65™ Cleaner предназначен для всех типов пластиковых трубопроводов из НПВХ, ХПВХ, АБС и Стирола. Он специально разработан для удаления жира, масел и грязи с соединяемых при склеивании поверхностей пластиковых трубопроводов. Должен быть использован только в качестве очистителя, в сочетании с подходящим сочетанием грунтовки и клея. Также может использоваться в качестве очистителя для инструмента.

  • Пластик НПВХ / ХПВХ / АБС /Стирол
  • Применение обезжиривание поверхностей, очистка инструмента
  • Цвет прозрачный

Прекрасный клей

Экологически чистый клей для ПВХ труб Weld On 705 ECO с ультранизким содержанием летучих органических веществ (ЛОВ). Клей для ПВХ 705 ECO™ подходит для всех видов жестких труб. Отсутствие в составе клея тетрагидрофурана создает благоприятные условия монтажа внутри помещений. Небольшая вязкость клея сокращает его расход. Клей для ПВХ труб 705 ECO рекомендуется использовать для диаметров до 160 мм. Клей для ПВХ труб Weld On 705 ECO™ произведен в США.
